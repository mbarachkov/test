World View

Web app for looking up news and images from any destination

An app for created for travel. Look up recent, local images as well as lookup of local, current news articles pertaining to your lookup.

This project was a collaboration of 4 Boot Camp students with the use of HTML, CSS and JavaScript as our front end. We used Node to setup firebase as the platform for our back end.
